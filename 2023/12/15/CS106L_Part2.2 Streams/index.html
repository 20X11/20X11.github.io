##what a stream is

be able to interact with external devices (console 2 keyboard)

write files

program 2 program (pipelines) (CS 110)

##string stream

not tied to anything , only does conversion itself

construct a string stream (input & output)

you can change the position point starts at the end

now it’s an input stream

stream can auto convert data type

when reading a string , dilimeter is a space key.

when you want to define dilimeter by yourself:

>> extracts the next variable of a certain type,up to the next whitespace;

The >> and << operators return a reference to the stream itself, so in each instance the stream is the left-hand operand.

stream positioning functions

and iss.tellg() is also this function

easy function

four bits indicate the state of the stream

  • Good bit : ready for read/write

  • Fail bit : previous operation failed, all future operations frozen.

  • EOF bit : previous operation reached the end of buffer content.

  • Bad bit : external error, likely irrecoverable.
